## 🔍 What it checks

ai-quality-gate scans your code and looks for issues in these areas:

- **ESLint** — finds code patterns that can cause bugs or confusion
- **TypeScript** — checks type problems before they reach users
- **Prettier** — helps keep code formatting clean and even
- **SonarQube-style checks** — finds weak spots in code quality
- **Security checks** — flags risky code patterns
- **Complexity checks** — spots code that may be too hard to follow
